    Type Alias SqlDialect

    Owns everything that differs syntactically between database engines, so the query builders below (whereClause, mkWhere, ...) never branch on the engine. Each driver package (postgres, sqlite, mysql-backend, ...) provides one.

    is_sqlite is kept because many call sites use it as a proxy for "no schema/tenant prefix" (true only for sqlite).
