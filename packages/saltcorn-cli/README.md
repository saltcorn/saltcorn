# Usage

<!-- usage -->
```sh-session
$ npm install -g @saltcorn/cli
$ saltcorn COMMAND
running command...
$ saltcorn (-v|--version|version)
@saltcorn/cli/0.3.4-beta.0 linux-x64 node-v14.4.0
$ saltcorn --help [COMMAND]
USAGE
  $ saltcorn COMMAND
...
```

## `saltcorn backup`

Backup the PostgreSQL database to a file with pg_dump

```
USAGE
  $ saltcorn backup

OPTIONS
  -o, --output=output  [default: 20210121-saltcorn-LAPTOP-3VET16LK.sqlc] output filename
```

_See code: [src/commands/backup.js](https://github.com/saltcorn/saltcorn/blob/v0.3.4-beta.0/src/commands/backup.js)_
