# Usage

<!-- usage -->

```sh-session
$ npm install -g saltcorn-cli
$ saltcorn COMMAND
running command...
$ saltcorn (-v|--version|version)
saltcorn-cli/0.0.1 linux-x64 node-v12.16.1
$ saltcorn --help [COMMAND]
USAGE
  $ saltcorn COMMAND
...
```
